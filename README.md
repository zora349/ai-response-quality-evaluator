# AI Response Evaluation Project

## 项目简介

本项目是一个用于评估 AI 回复质量的自动化分析项目，主要用于模拟 AI 训练师 / AI 质量评估岗位中的真实工作流程。

项目通过读取包含用户问题、模型回复、人工标注结果的数据集，对 AI 回复进行多维度质量分析，并自动生成 Excel 评估结果和 Markdown 错误分析报告。

该项目重点关注：

- AI 回复是否满足用户需求
- 是否存在指令违背
- 是否存在回答不完整
- 是否存在事实错误
- 是否存在格式问题
- 是否存在表达不自然等问题

---

## 项目目标

本项目的目标是搭建一个基础的 AI 回复质量评估流程，完成从数据读取、规则判断、结果统计到报告生成的完整闭环。

主要目标包括：

1. 读取待评估的 AI 回复数据
2. 根据评价规则判断回复是否合格
3. 统计合格率、平均分、错误类型分布
4. 输出 Excel 格式的评估结果
5. 自动生成 Markdown 错误分析报告
6. 通过图表展示评估结果，辅助复盘与优化

评估维度

本项目主要从以下维度评估 AI 回复质量：

评估维度	说明
指令遵循	是否完全按照用户要求完成任务
内容完整性	是否遗漏用户要求的关键信息
事实准确性	是否存在明显事实错误
格式符合度	是否符合用户指定格式
表达质量	语言是否自然、清晰、易理解
安全合规	是否存在不适当或高风险内容
错误类型

项目中常见错误类型包括：

错误类型	说明
指令违背	没有遵守用户明确限制
回答不完整	遗漏了用户要求的部分内容
格式错误	输出格式不符合要求
事实错误	回答中存在明显错误信息
表达问题	表达不清晰、不自然或不符合语境
无明显错误	回复整体符合要求
