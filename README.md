# AI 回复质量自动评估项目

## 一、项目简介

本项目是一个面向 AI 训练师 / AI 质检岗位的 **AI 回复质量自动评估工具**。

项目通过 Python 对 AI 回复样本进行批量评估，自动判断模型回复是否满足用户指令，并输出评分结果、错误类型、质量分析报告和可视化图表。

项目主要用于模拟 AI 训练师在实际工作中的质量评估流程，包括：

* 判断模型回复是否合格
* 识别常见问题类型
* 统计整体合格率和平均分
* 输出 Excel 多 Sheet 分析结果
* 生成 Markdown 汇总报告
* 生成基础可视化图表

本项目从原始单脚本版本重构为模块化结构，提升了代码可维护性，也方便后续扩展更多评估规则、DeepEval 自动评估或不同数据源。

---

## 二、项目目标

本项目主要解决以下问题：

1. **提升 AI 回复质检效率**
   通过自动化规则评估，减少人工逐条检查的重复工作。

2. **沉淀可复用的评估流程**
   将数据读取、规则判断、评分、错误分类、报告生成拆分为独立模块。

3. **辅助定位模型常见问题**
   通过错误类型统计和低分样本分析，快速发现模型回复中的主要问题。

4. **形成可展示的项目成果**
   输出 Excel 报告、Markdown 报告和图表，便于项目复盘和面试展示。

## 四、核心功能

### 1. 数据读取与字段检查

通过 `data_loader.py` 读取待评估数据集，并对必要字段进行基础检查，确保后续评估流程可以正常运行。

主要功能包括：

* 读取 CSV 样本数据
* 检查必要字段是否存在
* 返回可用于评估的 DataFrame

---

### 2. 规则提取与规则判断

通过 `evaluator.py` 对用户指令进行规则分析，识别模型回复需要满足的要求。

支持的规则类型包括：

* 数量要求
* 字数限制
* 格式要求
* 禁止词限制
* 语言风格要求
* 内容完整性要求

---

### 3. 自动评分与合格判断

项目会根据规则检查结果，对每条 AI 回复进行自动评分，并判断回复是否合格。

评估结果包括：

* 自动评分
* 合格 / 不合格结论
* 错误类型
* 评价理由
* 综合质量分

---

### 4. 错误类型分类

项目会根据评估结果自动识别常见错误类型，例如：

* 指令违背
* 回答不完整
* 格式错误
* 内容不准确
* 表达不自然
* 无明显错误

这部分可用于模拟 AI 训练师在实际质检工作中的错误归因能力。

---

### 5. Excel 多 Sheet 分析报告

通过 `report.py` 自动生成 Excel 分析结果，包含多个 Sheet：

1. 汇总
2. 错误类型统计
3. 评分维度统计
4. 低分样本
5. 明细数据

Excel 报告便于进一步查看样本明细、筛选低分回复、分析错误分布。

---

### 6. Markdown 汇总报告

通过 `report_generator.py` 自动生成 AI 回复质量自动评估报告，内容包括：

* 总样本数
* 合格数量
* 不合格数量
* 合格率
* 综合平均分
* 错误类型统计
* 质量分析总结

Markdown 报告适合用于项目复盘、GitHub 展示和面试说明。

---

### 7. 可视化图表输出

项目会根据评估结果生成基础可视化图表，例如：

* 合格 / 不合格分布图
* 错误类型统计图
* 评分维度分布图

图表可用于快速查看模型回复质量的整体情况。

## 九、数据字段说明

样本数据存放在 `data/auto_eval_samples.csv` 中。

数据集主要包含以下类型字段：

| 字段类型 | 说明                |
| ---- | ----------------- |
| 用户指令 | 用户提出的原始需求         |
| 模型回复 | 待评估的 AI 回复内容      |
| 参考信息 | 用于辅助判断的标准或参考内容    |
| 规则字段 | 从用户指令中提取出的限制条件    |
| 评估结果 | 自动评分、结论、错误类型和评价理由 |

实际字段名称以项目数据集中的列名为准。

## 十、模块说明

### `config.py`

负责项目路径和基础配置，例如：

* 数据文件路径
* 输出文件路径

### `data_loader.py`

负责读取数据集，并完成基础字段检查。

---

### `evaluator.py`

项目核心评估模块，负责：

* 提取用户指令中的规则
* 判断模型回复是否满足规则
* 计算评分
* 判断合格 / 不合格
* 分类错误类型
* 批量评估数据集

---

### `report.py`

负责生成评估后的结果文件，包括：

* Excel 多 Sheet 分析报告
* Markdown 明细内容
* 可视化图表
* 基础样式美化

---

### `report_generator.py`

负责生成汇总型 AI 回复质量自动评估报告，包括：

* 总体合格率
* 平均分
* 错误类型统计
* 质量分析总结

---

### `main.py`

项目主入口，负责串联完整流程：

```text
读取数据 → 执行评估 → 合并结果 → 生成报告 → 输出统计信息
```

---

### `run_deepeval.py`

DeepEval 自动评估入口，用于后续扩展模型自动评估能力。

---

### `upgrade_dataset.py`

用于对原始数据集进行字段补充、结构升级或样本整理。
